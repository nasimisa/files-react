## Prerequisites

Before you begin, ensure you have the following installed:

- **Node.js**: >= 18.18.0
- **pnpm**: >= 10.13.1

## Getting Started

1. **Clone or navigate to the project directory**

   ```bash
   cd files-react-assessment
   ```

2. **Install dependencies**

   ```bash
   pnpm install
   ```

3. **Start the development server**

   ```bash
   pnpm dev
   ```

   The application will automatically open in your browser at `http://localhost:3000` (or the next available port).

4. **Build for production**

   ```bash
   pnpm build
   ```

5. **Preview production build**
   ```bash
   pnpm preview
   ```
